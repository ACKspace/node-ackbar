# requirements
node >= 12.2.0
npm >= 6.9.0

Packages:
* figlet
* serialport

# How to setup your system
install node and npm
`sudo apt install node`

# install "n" package
`sudo npm install -g n`

# update to latest node
`n latest`

# install packages
`npm install figlet serialport`

# run ACKbar
node finalbar
